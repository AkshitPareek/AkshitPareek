## <img src="https://media.giphy.com/media/iY8CRBdQXODJSCERIr/giphy.gif" width="24"> Experience

### <img src="https://img.shields.io/badge/Texas%20Instruments-CC0000?style=flat&logo=texas-instruments&logoColor=white" /> Texas Instruments — TI Deep Learning
**Senior Software Engineer** | Bengaluru, India | *Feb 2025 – Present*

- Optimizing ML inference kernels on **TI C7x DSPs (SIMD)** across production workloads — focusing on vectorization, memory bandwidth, and numerical stability
- Implemented **40+ AI operators** (Softmax, LayerNorm, BatchNorm, GeLU, etc.) in **BF16** and **FP8 (e4m3)**, matching fixed-point latency through SIMD and instruction-level parallelism
- Led bring-up of a **new SDK** for kernel unit testing and reusable inference/runtime library
- Designed **parallel Ninja build pipelines** leveraging Amdahl's and Gustafson's scaling principles
- Created **PyTorch golden reference suite for 70+ operators** — verification baseline across kernel teams
- Built internal dev tooling: **ISA ↔ intrinsic mapping database**, kernel scaffolding generators, automated validation pipelines
- Architected **Energy Metrology library on C2000 MCUs** for data center power quality analysis — sampling at 8/16 ksps, achieving **0.002% THD error**

### <img src="https://img.shields.io/badge/Texas%20Instruments-CC0000?style=flat&logo=texas-instruments&logoColor=white" /> Texas Instruments — ASM Systems
**Software Engineer** | Bengaluru, India | *Aug 2023 – Feb 2025*

- Built **multi-core Traction Inverter reference solution** for ARM Cortex-R based MCUs
- Enabled **G3 Power Line Communication (PLC)** on C2000 Gen-2 devices for solar farm applications
- Engineered **software-based multi-channel SPI** using GPIO on AM26x (up to **10 MHz**), eliminating hardware SPI limits
- Maintained drivers for analog/control peripherals: **ADC, EPWM, EQEP, DAC, ECAP**
- Real-time systems development support for industrial & automotive clients

### <img src="https://img.shields.io/badge/Innovate5G-00A4EF?style=flat" /> Innovate5G
**Software Engineering Intern** | Dallas, TX (Remote) | *Jan 2022 – Jul 2022**

- Built multi-threaded embedded software for **4G/5G wireless observability hardware dongle**
- Delivered working prototype in 6 months within a 2-person team
